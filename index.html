System Snapshot Webserver.

Collect the computer and system info at a given time in order to track issues or problems. You can share the generated report to help you detect your computer problem at a given moment. Works on Windows / Linux OS.

What kind of Reports

  1. System Info
  2. Hardware
  3. Networks
  4. PnP Devices
  5. Processes
  6. Services
  7. Benchmark
  8. Installed programs
  9. Special files

How to use

Run program (.exe) as Administrator. Wait util report gets generated. Open or send the report to someone who can read it and analyse or problem.

Note: Program can be executed with '-s', '/s' or '--silent' argument to skip GUI. Report will be generated and the html file will show on explorer after completion.

Example: "SystemInfoSnapshot.exe -s"
